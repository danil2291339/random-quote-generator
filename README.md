## Описание программы

**Random Quote Generator** — это десктопное приложение с графическим интерфейсом для генерации вдохновляющих цитат. Программа позволяет:

- Генерировать случайные цитаты из предустановленной коллекции
- Добавлять собственные цитаты
- Просматривать историю всех сгенерированных цитат
- Фильтровать историю по автору и теме
- Сохранять историю и базу цитат в JSON-файлы

## Как пользоваться

### Генерация цитаты
Нажмите кнопку **«✨ Сгенерировать цитату»** — случайная цитата появится в верхней части окна.

### История цитат
Все сгенерированные цитаты автоматически сохраняются в таблице **«История цитат»** с указанием даты и времени.

### Фильтрация
Выберите автора или тему в выпадающих списках, чтобы отфильтровать историю. Кнопка **«Сбросить фильтры»** возвращает полный список.

### Добавление цитаты
Заполните поля **«Текст»**, **«Автор»** и **«Тема»** в нижней части окна и нажмите **«💾 Сохранить цитату»**. Новая цитата появится в общей коллекции.

## Пример работы

### Генерация цитаты
При нажатии на кнопку генерации появляется:

```
❝ Будь изменением, которое ты хочешь видеть в мире. ❞
— Махатма Ганди
📌 Тема: Мотивация
```

### Фильтрация по автору
1. Выберите **«Махатма Ганди»** в фильтре «Фильтр по автору»
2. В истории останутся только цитаты этого автора

### Добавление новой цитаты

| Поле | Значение |
|------|----------|
| Текст | Программирование — это искусство объяснять компьютеру, что ты хочешь |
| Автор | Аноним |
| Тема | IT |

После нажатия **«Сохранить»** цитата доступна для генерации.

## Тестирование

### Позитивные тесты

| Тест | Действие | Ожидаемый результат |
|------|----------|---------------------|
| Генерация цитаты | Нажать кнопку генерации | Появляется цитата, добавляется в историю |
| Добавление цитаты | Заполнить все поля и сохранить | Цитата появляется в списке доступных |
| Фильтрация | Выбрать автора | История показывает только его цитаты |

### Негативные тесты

| Тест | Действие | Ожидаемый результат |
|------|----------|---------------------|
| Пустой текст | Оставить поле «Текст» пустым | Ошибка: «Все поля должны быть заполнены!» |
| Пустой автор | Оставить поле «Автор» пустым | Ошибка: «Все поля должны быть заполнены!» |
| Пустая тема | Оставить поле «Тема» пустым | Ошибка: «Все поля должны быть заполнены!» |

### Граничные тесты

| Тест | Действие | Результат |
|------|----------|-----------|
| Длинная цитата | Вставить цитату на 500 символов | ✅ Корректно отображается с переносом |
| Спецсимволы | Ввести цитату с кавычками, эмодзи | ✅ Корректно сохраняется в JSON |

## Данные автоматически сохраняются в JSON

### quotes.json (база цитат)
```json
[
  {
    "text": "Будь изменением, которое ты хочешь видеть в мире.",
    "author": "Махатма Ганди",
    "theme": "Мотивация"
  }
]
```

### history.json (история)
```json
[
  {
    "text": "Будь изменением, которое ты хочешь видеть в мире.",
    "author": "Махатма Ганди",
    "theme": "Мотивация",
    "timestamp": "2026-04-30 14:25:30"
  }
]
```
